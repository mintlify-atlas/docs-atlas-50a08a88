## Introduction

The Customer Management API allows you to create, retrieve, and manage customer records in the TechCore system. Customers are central to sales operations and are linked to sales transactions, payment plans, and account status tracking.

## Customer Data Model

Customers in TechCore are stored in the `clientes` table and contain the following information:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `codclien` | string | Yes | Unique customer code (primary key, max 50 characters) |
| `nombre` | string | Yes | Customer full name (max 200 characters) |
| `telefono` | string | No | Phone number (max 15 characters) |
| `email` | string | No | Email address (max 200 characters) |
| `direccion` | string | No | Physical address (max 300 characters) |
| `estado` | boolean | No | Active status (default: true) |
| `created_date` | datetime | No | Record creation timestamp (auto-generated) |

## Database Indexes

The customers table is optimized with indexes on:
- **Email**: Fast lookup by email address
- **Estado**: Quick filtering of active/inactive customers
- **Nombre**: Efficient customer name searches

## Common Use Cases

### New Customer Registration
Create a customer record before processing their first sale. The `codclien` field serves as the unique identifier throughout the system.

### Customer Lookup
Search for customers by name, email, or customer code to retrieve their information and sales history.

### Account Management
Monitor customer account status including outstanding balances, payment plans, and credit utilization.

### Customer Status Control
Activate or deactivate customer accounts using the `estado` field without deleting records, preserving historical data.
