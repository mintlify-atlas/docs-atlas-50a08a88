## Overview

The Categories API enables management of product categories, allowing you to organize products into logical groups for better inventory organization.

## Category Model

The category model contains the following fields:

<ResponseField name="codCategoria" type="integer" required>
  Unique category identifier (Primary Key, auto-increment)
</ResponseField>

<ResponseField name="codigo" type="string" required>
  Unique category code (max 20 characters)
</ResponseField>

<ResponseField name="nombre" type="string" required>
  Category name (max 150 characters)
</ResponseField>

<ResponseField name="descripcion" type="string">
  Category description (max 300 characters)
</ResponseField>

<ResponseField name="estado" type="boolean">
  Category active status (default: true)
</ResponseField>

<ResponseField name="created_date" type="datetime">
  Record creation timestamp (auto-generated)
</ResponseField>

## Get All Categories

<ParamField query="estado" type="boolean">

## Create Category

<api-endpoint api="POST" path="/api/categories" />

Creates a new product category.

<ParamField body="codigo" type="string" required>
  Unique category code (max 20 characters)
</ParamField>

<ParamField body="nombre" type="string" required>
  Category name (max 150 characters)
</ParamField>

<ParamField body="descripcion" type="string">
  Category description (max 300 characters)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.techcore.com/api/categories" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "codigo": "CAT-004",
    "nombre": "Impresoras",
    "descripcion": "Impresoras láser e inkjet"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/categories', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    codigo: 'CAT-004',
    nombre: 'Impresoras',
    descripcion: 'Impresoras láser e inkjet'
  })
});
```
</RequestExample>

<ResponseExample>
```json 201 - Created
{
  "codCategoria": 4,
  "codigo": "CAT-004",
  "nombre": "Impresoras",
  "descripcion": "Impresoras láser e inkjet",
  "estado": true,
  "created_date": "2026-03-04T15:30:00Z"
}
```

```json 400 - Bad Request
{
  "error": "Category code already exists",
  "codigo": "CAT-004"
}
```

```json 400 - Validation Error
{
  "error": "Validation failed",
  "details": [
    {
      "field": "nombre",
      "message": "Category name is required"
    }
  ]
}
```
</ResponseExample>

## Delete Category

<api-endpoint api="DELETE" path="/api/categories/{codCategoria}" />

Deletes a category (soft delete by setting estado to false). Cannot delete if products exist in the category.

<ParamField path="codCategoria" type="integer" required>
  Category identifier to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.techcore.com/api/categories/4" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/categories/4', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "message": "Category deleted successfully",
  "codCategoria": 4
}
```

```json 400 - Conflict
{
  "error": "Cannot delete category with associated products",
  "codCategoria": 4,
  "productCount": 5
}
```

```json 404 - Not Found
{
  "error": "Category not found",
  "codCategoria": 4
}
```
</ResponseExample>
