## Overview

The Products API allows you to manage product inventory including creation, retrieval, updates, and deletion of products in the TechCore system.

## Product Model

The product model contains the following fields:

<ResponseField name="codprod" type="string" required>
  Unique product code identifier (Primary Key)
</ResponseField>

<ResponseField name="codCategoria" type="integer">
  Category code linking to the product's category
</ResponseField>

<ResponseField name="descripcion" type="string">
  Product description (max 500 characters)
</ResponseField>

<ResponseField name="precioCompra" type="decimal" required>
  Purchase price (decimal 18,2)
</ResponseField>

<ResponseField name="precioVenta" type="decimal" required>
  Sale price (decimal 18,2)
</ResponseField>

<ResponseField name="stock" type="integer">
  Current stock quantity (default: 0)
</ResponseField>

<ResponseField name="stockMinimo" type="integer">
  Minimum stock threshold (default: 5)
</ResponseField>

<ResponseField name="estado" type="boolean">
  Product active status (default: true)
</ResponseField>

<ResponseField name="created_date" type="datetime">
  Record creation timestamp (auto-generated)
</ResponseField>

## Create Product

<api-endpoint api="POST" path="/api/products" />

Creates a new product in the inventory.

<ParamField body="codprod" type="string" required>
  Unique product code
</ParamField>

<ParamField body="codCategoria" type="integer">
  Category identifier
</ParamField>

<ParamField body="descripcion" type="string">
  Product description
</ParamField>

<ParamField body="precioCompra" type="decimal" required>
  Purchase price
</ParamField>

<ParamField body="precioVenta" type="decimal" required>
  Sale price
</ParamField>

<ParamField body="stock" type="integer">
  Initial stock quantity
</ParamField>

<ParamField body="stockMinimo" type="integer">
  Minimum stock threshold
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.techcore.com/api/products" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "codprod": "PROD003",
    "codCategoria": 1,
    "descripcion": "Teclado Mecánico RGB",
    "precioCompra": 35.00,
    "precioVenta": 59.99,
    "stock": 20,
    "stockMinimo": 5
  }'
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/products', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    codprod: 'PROD003',
    codCategoria: 1,
    descripcion: 'Teclado Mecánico RGB',
    precioCompra: 35.00,
    precioVenta: 59.99,
    stock: 20,
    stockMinimo: 5
  })
});
```
</RequestExample>

<ResponseExample>
```json 201 - Created
{
  "codprod": "PROD003",
  "codCategoria": 1,
  "descripcion": "Teclado Mecánico RGB",
  "precioCompra": 35.00,
  "precioVenta": 59.99,
  "stock": 20,
  "stockMinimo": 5,
  "estado": true,
  "created_date": "2026-03-04T14:25:00Z"
}
```

```json 400 - Bad Request
{
  "error": "Product code already exists",
  "codprod": "PROD003"
}
```
</ResponseExample>

## Delete Product

<api-endpoint api="DELETE" path="/api/products/{codprod}" />

Deletes a product from the inventory (soft delete by setting estado to false).

<ParamField path="codprod" type="string" required>
  Product code to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.techcore.com/api/products/PROD003" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/products/PROD003', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "message": "Product deleted successfully",
  "codprod": "PROD003"
}
```

```json 404 - Not Found
{
  "error": "Product not found",
  "codprod": "PROD003"
}
```
</ResponseExample>
