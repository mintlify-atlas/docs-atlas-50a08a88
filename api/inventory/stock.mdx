## Overview

The Stock Management API provides endpoints for tracking and managing product inventory levels, including stock adjustments, low-stock alerts, and stock movement history.

## Stock Operations

### Get Current Stock

<api-endpoint api="GET" path="/api/stock/{codprod}" />

Retrieves current stock information for a specific product.

<ParamField path="codprod" type="string" required>
  Product code identifier
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.techcore.com/api/stock/PROD001" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/stock/PROD001', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "codprod": "PROD001",
  "descripcion": "Laptop HP 15-dy2021la",
  "stock": 15,
  "stockMinimo": 5,
  "stockStatus": "NORMAL",
  "availableForSale": true,
  "lastUpdated": "2026-03-04T10:30:00Z"
}
```

```json 404 - Not Found
{
  "error": "Product not found",
  "codprod": "PROD001"
}
```
</ResponseExample>

### Update Stock (Adjust)

<api-endpoint api="POST" path="/api/stock/adjust" />

Manually adjusts stock levels for a product. Use for inventory corrections, damages, or manual adjustments.

<ParamField body="codprod" type="string" required>
  Product code identifier
</ParamField>

<ParamField body="quantity" type="integer" required>
  Quantity to adjust (positive to add, negative to subtract)
</ParamField>

<ParamField body="reason" type="string" required>
  Reason for adjustment (e.g., "CORRECTION", "DAMAGE", "THEFT", "MANUAL")
</ParamField>

<ParamField body="notes" type="string">
  Additional notes about the adjustment
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.techcore.com/api/stock/adjust" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "codprod": "PROD001",
    "quantity": -2,
    "reason": "DAMAGE",
    "notes": "Productos dañados durante transporte"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/stock/adjust', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    codprod: 'PROD001',
    quantity: -2,
    reason: 'DAMAGE',
    notes: 'Productos dañados durante transporte'
  })
});
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "codprod": "PROD001",
  "previousStock": 15,
  "adjustment": -2,
  "newStock": 13,
  "stockMinimo": 5,
  "reason": "DAMAGE",
  "timestamp": "2026-03-04T14:45:00Z"
}
```

```json 400 - Invalid Adjustment
{
  "error": "Insufficient stock for adjustment",
  "codprod": "PROD001",
  "currentStock": 3,
  "requestedAdjustment": -5
}
```
</ResponseExample>

### Bulk Stock Update

<api-endpoint api="POST" path="/api/stock/bulk-update" />

Updates stock levels for multiple products at once. Useful for physical inventory counts.

<ParamField body="updates" type="array" required>
  Array of stock update objects
</ParamField>

<ParamField body="updates[].codprod" type="string" required>
  Product code
</ParamField>

<ParamField body="updates[].newStock" type="integer" required>
  New stock level
</ParamField>

<ParamField body="reason" type="string" required>
  Reason for bulk update (e.g., "PHYSICAL_COUNT", "INVENTORY_AUDIT")
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.techcore.com/api/stock/bulk-update" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "updates": [
      {"codprod": "PROD001", "newStock": 20},
      {"codprod": "PROD002", "newStock": 45},
      {"codprod": "PROD003", "newStock": 12}
    ],
    "reason": "PHYSICAL_COUNT"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/stock/bulk-update', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    updates: [
      { codprod: 'PROD001', newStock: 20 },
      { codprod: 'PROD002', newStock: 45 },
      { codprod: 'PROD003', newStock: 12 }
    ],
    reason: 'PHYSICAL_COUNT'
  })
});
```
</RequestExample>

## Stock Minimum Thresholds

### Update Stock Minimum

<api-endpoint api="PUT" path="/api/stock/{codprod}/minimum" />

Updates the minimum stock threshold for a product.

<ParamField path="codprod" type="string" required>
  Product code identifier
</ParamField>

<ParamField body="stockMinimo" type="integer" required>
  New minimum stock threshold
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT "https://api.techcore.com/api/stock/PROD001/minimum" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"stockMinimo": 10}'
```

```javascript JavaScript
const response = await fetch('https://api.techcore.com/api/stock/PROD001/minimum', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    stockMinimo: 10
  })
});
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "codprod": "PROD001",
  "previousMinimum": 5,
  "newMinimum": 10,
  "currentStock": 13,
  "status": "NORMAL",
  "message": "Stock minimum updated successfully"
}
```

```json 400 - Warning
{
  "codprod": "PROD001",
  "newMinimum": 10,
  "currentStock": 8,
  "status": "LOW",
  "warning": "Current stock is below new minimum threshold"
}
```
</ResponseExample>
