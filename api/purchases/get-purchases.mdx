## Get All Purchases

```
GET /api/purchases
```

Retrieves a list of all purchase orders in the system with optional filtering.

### Query Parameters

<ParamField query="codprov" type="string">
  Filter purchases by supplier code.
</ParamField>

<ParamField query="codusu" type="integer">
  Filter purchases by user ID who created the order.
</ParamField>

<ParamField query="estado" type="integer">
  Filter by purchase order status (1 = active, 0 = inactive).
</ParamField>

<ParamField query="fechaInicio" type="date">
  Filter purchases from this date onwards (inclusive).
</ParamField>

<ParamField query="fechaFin" type="date">
  Filter purchases up to this date (inclusive).
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="pageSize" type="integer" default="20">
  Number of records per page.
</ParamField>

<ParamField query="includeDetails" type="boolean" default="false">
  Include purchase detail lines in the response.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL - All Purchases
curl -X GET "https://api.techcore.com/api/purchases" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash cURL - Filtered
curl -X GET "https://api.techcore.com/api/purchases?codprov=PROV-001&estado=1&includeDetails=true" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash cURL - Date Range
curl -X GET "https://api.techcore.com/api/purchases?fechaInicio=2026-01-01&fechaFin=2026-03-31" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

## Get Purchase by ID

```
GET /api/purchases/{norden}
```

Retrieves a specific purchase order by its order number.

### Path Parameters

<ParamField path="norden" type="string" required>
  The unique purchase order number to retrieve.
</ParamField>

### Query Parameters

<ParamField query="includeDetails" type="boolean" default="true">
  Include purchase detail lines in the response. Defaults to true for single purchase retrieval.
</ParamField>

<ParamField query="includeSupplier" type="boolean" default="false">
  Include supplier information in the response.
</ParamField>

<ParamField query="includeUser" type="boolean" default="false">
  Include user information in the response.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.techcore.com/api/purchases/PO-2026-0001" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash cURL - With Related Data
curl -X GET "https://api.techcore.com/api/purchases/PO-2026-0001?includeSupplier=true&includeUser=true" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

## Database Indexes

The following indexes are available to optimize query performance:

- `IDX_compras_codprov` - Filter by supplier
- `IDX_compras_codusu` - Filter by user
- `IDX_compras_estado` - Filter by status
- `IDX_compras_fecha` - Filter by date range

Use these indexed fields in your queries for better performance.
