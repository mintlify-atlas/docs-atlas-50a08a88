## Overview

Purchase details represent individual line items within a purchase order. Each detail line specifies a product, quantity, unit price, and line subtotal.

## Get Purchase Details

```
GET /api/purchases/{norden}/details
```

Retrieves all detail lines for a specific purchase order.

### Path Parameters

<ParamField path="norden" type="string" required>
  The purchase order number to retrieve details for.
</ParamField>

### Query Parameters

<ParamField query="includeProduct" type="boolean" default="false">
  Include product information for each detail line.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.techcore.com/api/purchases/PO-2026-0001/details" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```bash cURL - With Product Info
curl -X GET "https://api.techcore.com/api/purchases/PO-2026-0001/details?includeProduct=true" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Get Single Detail Line

```
GET /api/purchases/details/{id}
```

Retrieves a specific purchase detail line by its ID.

### Path Parameters

<ParamField path="id" type="integer" required>
  The unique ID of the detail line to retrieve.
</ParamField>

```bash cURL
  -H "Authorization: Bearer YOUR_TOKEN"
```

</CodeGroup>

## Add Detail Line

```
POST /api/purchases/{norden}/details
```

Adds a new detail line to an existing purchase order.

### Path Parameters

<ParamField path="norden" type="string" required>
  The purchase order number to add the detail line to.
</ParamField>

### Request Body

<ParamField body="codprod" type="string" required>
  Product code. Must reference an existing product. Maximum 50 characters.
</ParamField>

<ParamField body="cantidad" type="integer" required>
  Quantity of the product. Must be greater than 0.
</ParamField>

<ParamField body="precio" type="decimal" required>
  Unit price for the product. Precision: 18 digits, 2 decimal places.
</ParamField>

<ParamField body="subtotal" type="decimal" required>
  Line subtotal (cantidad × precio). Precision: 18 digits, 2 decimal places.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X POST "https://api.techcore.com/api/purchases/PO-2026-0001/details" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "codprod": "PROD-003",
    "cantidad": 20,
    "precio": 50.00,
    "subtotal": 1000.00
  }'
```

```json JSON
{
  "codprod": "PROD-003",
  "cantidad": 20,
  "precio": 50.00,
  "subtotal": 1000.00
}
```
</CodeGroup>

### Response Example

<CodeGroup>
```json 201 Created
{
  "id": 3,
  "norden": "PO-2026-0001",
  "codprod": "PROD-003",
  "cantidad": 20,
  "precio": 50.00,
  "subtotal": 1000.00
}
```

```json 400 Bad Request
{
  "error": "Bad Request",
  "message": "Invalid detail line data",
  "details": [
    "cantidad must be greater than 0",
    "subtotal does not match cantidad × precio"
  ]
}
```

```json 404 Not Found
{
  "error": "Not Found",
  "message": "Purchase order 'PO-2026-0001' not found"
}
```
</CodeGroup>

## Update Detail Line

```
PUT /api/purchases/details/{id}
```

Updates an existing purchase detail line.

### Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the detail line to update.
</ParamField>

### Request Body

<ParamField body="cantidad" type="integer">
  Updated quantity. Must be greater than 0.
</ParamField>

<ParamField body="precio" type="decimal">
  Updated unit price. Precision: 18 digits, 2 decimal places.
</ParamField>

<ParamField body="subtotal" type="decimal">
  Updated line subtotal. Precision: 18 digits, 2 decimal places.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X PUT "https://api.techcore.com/api/purchases/details/1" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "cantidad": 15,
    "precio": 95.00,
    "subtotal": 1425.00
  }'
```
</CodeGroup>

## Delete Detail Line

```
DELETE /api/purchases/details/{id}
```

Deletes a purchase detail line. Note: You cannot delete the last detail line from a purchase order.

### Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the detail line to delete.
</ParamField>

### Request Example

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.techcore.com/api/purchases/details/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

### Response Example

<CodeGroup>
```json 204 No Content
```

```json 400 Bad Request
{
  "error": "Bad Request",
  "message": "Cannot delete the last detail line from a purchase order"
}
```

```json 404 Not Found
{
  "error": "Not Found",
  "message": "Purchase detail with ID 1 not found"
}
```
</CodeGroup>

## Database Schema

### ComprasDetalle Table

- **Table Name**: `comprasDetalle`
- **Primary Key**: `id` (int, auto-increment)
- **Foreign Keys**:
  - `norden` → compras.norden (CASCADE on delete)
  - `codprod` → productos.codprod (ClientSetNull on delete)
- **Indexes**:
  - `IDX_comprasDetalle_norden` - Optimizes queries by purchase order
  - `IDX_comprasDetalle_codprod` - Optimizes queries by product

## Business Rules

- Each purchase order must have at least one detail line
- The `codprod` must reference an existing product in the productos table
- The `norden` must reference an existing purchase order in the compras table
- Quantity must be greater than 0
- Line subtotal should equal `cantidad × precio`
- When a purchase order is deleted, all associated detail lines are automatically deleted (CASCADE)
- Products cannot be deleted if they are referenced in purchase detail lines
