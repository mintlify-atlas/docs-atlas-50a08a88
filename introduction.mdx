# Welcome to TechCore API

TechCore API is a .NET Core 10 Web API starter project for business management systems. The project includes a complete database schema for sales operations, purchase management, inventory control, and credit management, with the API controllers ready to be implemented.

## What is TechCore API?

TechCore API provides the **database foundation** for retail and distribution businesses that need to manage:

- **Sales Operations**: Database models for sales orders with support for both cash and credit transactions
- **Purchase Management**: Schema for supplier purchases and cost tracking
- **Inventory Control**: Product and category models with stock management support
- **Credit Management**: Payment plan models, installment tracking, and overdue payment views

Built with modern .NET Core 10, Entity Framework Core, and SQL Server, this project provides enterprise-grade database design with triggers, indexes, and views ready for implementation.

<Info>
**Development Status**: The database schema and entity models are fully implemented. API controllers need to be created to expose the business logic through RESTful endpoints. See the [API Reference](/api-reference) for implementation guidance.
</Info>

## Database Modules

The database includes complete schemas for all major business operations:

### Sales Schema

Entity models for complete sales workflows:
- **Venta**: Sales orders with tax calculation and payment tracking
- **VentasDetalle**: Line items for each sale
- **AbonosVenta**: Payment records with automatic balance updates
- **PlanPago**: Payment plans with installment schedules

### Purchases Schema

Models for supplier relationship management:
- **Compra**: Purchase orders with supplier tracking
- **ComprasDetalle**: Purchase line items with cost details
- **Proveedore**: Supplier information and contact details

### Inventory Schema

Product management with stock control:
- **Producto**: Product catalog with pricing and stock levels
- **Categorium**: Product categories and classifications
- **Triggers**: Automatic stock reduction on sales

### Credit Management Schema

Advanced payment and credit tracking:
- **PlanPago**: Installment schedules with due dates
- **Database Views**: Pre-built views for overdue tracking
- **Automatic Triggers**: Balance updates on payment receipt

### Advanced SQL Server Features

The database leverages enterprise capabilities:

- **Automated Triggers**: `TR_DisminuirStock` and `TR_ActualizarSaldo` for automatic updates
- **Optimized Indexes**: Strategic indexing on frequently queried fields
- **Database Views**: `vw_CuotasVencidas`, `vw_CuotasPorVencer`, `vw_EstadoCuenta`
- **Foreign Key Relationships**: Fully normalized schema with referential integrity

## Architecture Highlights

<Info>
  TechCore API is structured with a clean architecture pattern:
  - **Models**: Entity definitions and database context (✅ Complete)
  - **Interfaces**: Contract definitions for services (⏳ Ready for implementation)
  - **Concretes**: Service implementations and business logic (⏳ Ready for implementation)
  - **ViewModels**: Data transfer objects for API responses (⏳ Ready for implementation)
  - **Controllers**: RESTful endpoints (⏳ To be implemented)
</Info>

<Note>
  The API includes CORS support configured for cross-origin requests, making it easy to integrate with web and mobile clients. The root URL (`/`) automatically redirects to Swagger UI for immediate API exploration.
</Note>
